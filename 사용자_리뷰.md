# Jewel Box 사용자 리뷰 모음

> 로스트아크 공격대 운영 도구 **Jewel Box**를 실제로 쓰게 될 다양한 공대원들의
> 가상 사용자 페르소나와 사용 워크플로우, 그리고 각자의 솔직한 별점 리뷰입니다.
> 리뷰어는 모두 **개발 비전공자**이며, 오로지 게임(로스트아크) 사용 경험만을 기준으로 평가했습니다.

---

## 1. 김공대 (38) — 공격대장 / 우리 채널 운영자

**플레이 스타일:** 매주 공대 짜고 인원 관리하는 운영진. 가장 무겁게 쓰는 헤비 유저.

### 사용 워크플로우
1. 수요일 밤, 공대원들이 캐릭터 등록·갱신을 마쳤는지 **개인별 현황** 페이지에서 확인
2. **공대 편성** 들어가서 자동 편성 버튼 한 방 → 발탄/카멘/에키드나 파티가 아이템레벨·서폿 기준으로 자동으로 짜임
3. 빠지는 사람 생기면 **불참 관리**에서 누가 빠지면 어떤 공대가 펑크 나는지 미리 확인
4. 토벌 끝나면 **골드 정산**이 캐릭터별 상위 3개 자동 합산 → 주간 리셋 누르면 누적 골드로 넘어감

### 리뷰 ⭐️⭐️⭐️⭐️⭐️ (5/5)
> "솔직히 이거 없을 때 어떻게 살았나 싶다. 엑셀로 일일이 서폿 몇 명인지 세고, 누가 1710 넘었는지 확인하던 시절 생각하면 지옥이었음. **자동 편성이 서포터 부족까지 알아서 계산**해주는 게 제일 만족스럽다. 골드 정산도 상위 3개만 알아서 잡아주니까 계산 실수로 싸울 일이 없어졌다."
>
> **아쉬운 점:** 자동 편성이 가끔 내 의도랑 다르게 짜서 결국 수동으로 바꿀 때가 있다. 그리고 주간 리셋 버튼이 한 번 누르면 되돌릴 수 없는 것 같아서 누를 때마다 심장이 쫄깃하다. 실수 방지용 확인창이 좀 더 무겁게 떴으면 좋겠다.

## 2. 박복귀 (29) — 라이트 유저 / 주말 출석 공대원

**플레이 스타일:** 1년 쉬다 복귀. 게임 시스템 다 까먹음. 그냥 "나 언제 어디 가면 돼?"만 알고 싶음.

### 사용 워크플로우
1. 디스코드에 공유된 링크 클릭 → 폰으로 바로 접속
2. **일정 캘린더**만 보고 "아 이번 주 토요일 9시구나" 확인
3. 못 오는 날은 **불참 관리**에 체크만 하고 끝

### 리뷰 ⭐️⭐️⭐️⭐️ (4/5)
> "복귀하고 나서 제일 무서운 게 '나만 모르는 규칙' 있는 건데, 여기는 **캘린더만 봐도 일정이 한눈에** 들어와서 좋았다. 폰으로 앱처럼 설치도 되더라(이게 PWA라는 거라던데 뭔진 모름). 불참 체크도 한 번 누르면 끝이라 편하다."
>
> **아쉬운 점:** 처음 들어왔을 때 메뉴가 너무 많아서 어디부터 봐야 할지 막막했다. '개인별 현황', '공대 편성', '재련 견적' 이런 게 다 보이니까 나 같은 라이트 유저는 살짝 겁먹음. 복귀자용 '여기만 보세요' 같은 간단 모드가 있으면 별 다섯 줬을 듯.

## 3. 이부캐 (34) — 부캐 부자 / 원정대 6개 굴리는 유저

**플레이 스타일:** 본캐 1개 + 부캐 11개. 골드 수급 끝판왕. 캐릭터 관리가 인생.

### 사용 워크플로우
1. **내 원정대 관리** 모달에서 캐릭터 12개 한꺼번에 관리
2. 로스트아크 API **동기화** 버튼으로 아이템레벨·전투력 한 방에 업데이트
3. 부캐들을 별도 원정대(rosterId)로 묶어서 카제로스 같은 원정대 단위 레이드 대표 캐릭 지정
4. 귀속골드 받을 캐릭/안 받을 캐릭 옵션 설정

### 리뷰 ⭐️⭐️⭐️⭐️ (4/5)
> "캐릭 12개 관리하는 입장에서 **API 동기화 한 번에 아이템레벨이 싹 갱신**되는 게 진짜 신세계다. 일일이 손으로 1685, 1690 치던 거 생각하면... 부캐를 별도 원정대로 나눠서 관리하는 기능도 내 상황에 딱 맞았다. 귀속골드 옵션 세팅도 캐릭별로 잡을 수 있어서 정산이 깔끔하다."
>
> **아쉬운 점:** 캐릭이 많다 보니 관리 모달이 길어져서 스크롤 압박이 좀 있다. 그리고 동기화할 때 전투력이 떨어지는 건 안 올려주는데(이건 API 튕김 방지라곤 하지만), 가끔 진짜로 스펙 내린 캐릭은 수동으로 고쳐야 해서 헷갈렸다. 어떤 캐릭이 동기화 스킵됐는지 알림이 좀 더 눈에 띄었으면.

## 4. 최총무 (41) — 골드 정산 담당 / 우리 채널 살림꾼

**플레이 스타일:** 게임은 적당히, 대신 공금·정산은 칼같이. 골드 1원 단위까지 맞춰야 직성이 풀림.

### 사용 워크플로우
1. 매주 **개인별 현황**에서 공대원별 주간 골드 합산 확인
2. 일반골드/귀속골드 분리 정산 확인 (1710 넘는 레이드는 100% 일반, 이하는 50:50)
3. 주간 리셋 시 누적 골드로 자동 이월되는지 체크
4. 디스코드 봇으로 누가 안 깼는지 한 번 더 크로스체크

### 리뷰 ⭐️⭐️⭐️⭐️⭐️ (5/5)
> "정산 담당 입장에서 **골드 정산이 자동이라는 게 가장 큰 가치**다. 레이드별로 일반/귀속 비율이 패치 기준대로 자동 적용되니까 내가 표 만들 필요가 없어졌다. 과거 기록이 아이템레벨 바뀌어도 안 흔들리게 '그때 기록을 스냅샷으로 저장'한다는 점이 신뢰가 간다. 디스코드 봇이랑 숫자가 똑같이 나오는 것도 안심."
>
> **아쉬운 점:** 정산 로직이 좋은데, '이 골드가 왜 이렇게 계산됐는지' 풀어서 보여주는 상세 내역이 부족하다. 공대원이 "내 골드 왜 이래요?" 물어보면 내가 일일이 설명해야 해서, 클릭하면 계산 근거가 펼쳐지는 기능이 있으면 완벽하겠다.

## 5. 정모바일 (26) — 폰으로만 접속하는 유저

**플레이 스타일:** PC는 게임할 때만. 공대 관련 확인은 100% 폰. 출퇴근 지하철에서 체크.

### 사용 워크플로우
1. 폰 홈 화면에 추가한 Jewel Box 아이콘 터치(앱처럼 실행)
2. 다크모드로 **일정 캘린더**·**공대 편성** 확인
3. 심심하면 **경매 룰렛**, **사다리타기** 같은 미니게임으로 분배 정함

### 리뷰 ⭐️⭐️⭐️ (3/5)
> "폰에서 앱처럼 깔리고 **다크모드가 눈이 편한** 건 진짜 마음에 든다. 경매 아이템 룰렛이나 사다리타기 같은 소소한 게임도 공대원들이랑 '이거 누가 먹어' 정할 때 은근 재밌다. 디자인도 요즘 감성이라 깔끔하다."
>
> **아쉬운 점:** 그런데 폰으로 **공대 편성표는 좀 답답하다**. 파티가 가로로 길어서 좌우로 스크롤해야 하고, 표가 작아서 손가락으로 확대해야 보일 때가 있다. 모바일에선 카드 형태로 세로로 쭉 보여주면 훨씬 편할 듯. 게임은 만족, 표 보기는 불만이라 딱 중간 줬다.

## 6. 강신입 (22) — 갓 들어온 신규 공대원

**플레이 스타일:** 이 채널 들어온 지 일주일. 디스코드로 초대받아서 처음 접속.

### 사용 워크플로우
1. 디스코드에서 가토(봇) 명령어로 자기 캐릭 등록 시도
2. 웹에서 **내 원정대 관리**로 캐릭터 정보 입력
3. 본인이 어떤 레이드 자격이 되는지 **개인별 현황**에서 확인

### 리뷰 ⭐️⭐️⭐️ (3/5)
> "디스코드 봇으로 명령어 치면 응답이 바로 와서 '오 뭔가 잘 만들어진 채널이다' 싶었다. 내 아이템레벨로 갈 수 있는 레이드가 자동으로 걸러져서 보이는 것도 신입한텐 친절했다. 봇 응답이 전체 공개라 다른 사람들 어떻게 쓰는지 보고 따라 할 수 있던 것도 좋았음."
>
> **아쉬운 점:** 처음에 캐릭 등록을 디스코드로 해야 하는지 웹에서 해야 하는지 헷갈렸다. 안내가 없어서 선임한테 물어봤다. 신규용 가이드나 '시작하기' 한 페이지만 있어도 입문 장벽이 확 낮아질 것 같다. 기능은 좋은데 처음 진입이 불친절한 게 아쉬움.

## 7. 한도박 (31) — 미션 보드 / 주얼 베팅 애호가

**플레이 스타일:** 본 게임보다 공대 내 이벤트·내기를 더 즐기는 분위기 메이커.

### 사용 워크플로우
1. **미션 보드**에서 "이번 주 카멘 노데스 클리어" 같은 미션 생성
2. 공대원들끼리 성공/실패에 **주얼 베팅**
3. 미션 카드로 진행 현황 구경하며 채널 분위기 띄우기

### 리뷰 ⭐️⭐️⭐️⭐️⭐️ (5/5)
> "이거 진짜 신의 한 수다. **미션 만들고 베팅하는 재미**로 공대 출석률이 올랐다고 봐도 됨. '쫄작 노데스 가능?' 같은 거 걸어두면 다들 불타올라서 더 열심히 한다. 단순 일정 관리 툴인 줄 알았는데 **공대 분위기 살리는 게임 요소**까지 있는 게 차별점이다. 카드 디자인도 예뻐서 보는 맛이 있음."
>
> **아쉬운 점:** 미션 종류가 좀 더 다양했으면 한다. 그리고 지난 미션 결과가 쌓여서 '명예의 전당'처럼 통계로 보이면 더 불타오를 듯. 지금은 한 번 끝난 미션이 그냥 사라지는 느낌이라 아쉽다.

## 8. 윤스펙업 (35) — 재련·세팅 고민러

**플레이 스타일:** 골드 모이면 전부 재련에 박는 스펙업 중독자. 효율 계산이 취미.

### 사용 워크플로우
1. **재련 견적** 페이지에서 다음 단계 강화 비용 가늠
2. **고급 재련** 탭에서 재료·확률 비교
3. **경매 입찰 계산기**로 보석/장신구 적정 입찰가 계산
4. **채집** 도구로 생활 콘텐츠 수익 체크

### 리뷰 ⭐️⭐️⭐️⭐️ (4/5)
> "공대 툴인데 **재련 견적이랑 경매 계산기까지 들어있는 게** 예상 밖이었다. 강화 한 번 지르기 전에 비용 가늠하는 데 도움이 됐고, 경매 입찰가 계산기는 '얼마까지 불러야 손해 안 보나' 항상 헷갈렸는데 이걸로 깔끔해졌다. 하나의 사이트에서 정산도 하고 세팅 고민도 하니 효율적이다."
>
> **아쉬운 점:** 재련 견적의 계승 최저가 루트 같은 세부 계산은 아직 다듬어지는 중인 것 같다(데이터가 정확히 안 맞는 느낌). 그리고 이런 도구들이 메뉴 깊숙이 있어서 처음엔 있는 줄도 몰랐다. 강화 효율 자료는 외부 사이트도 많아서, 여기 데이터가 항상 최신 패치 기준인지 신뢰가 좀 더 쌓여야 메인으로 쓸 듯.

### 공통적으로 칭찬받은 점
- 자동 편성·골드 정산 등 **반복 작업의 자동화** (운영진·정산 담당 만족도 최상)

- 단순 관리 툴을 넘는 **미션 보드·미니게임** 등 분위기 요소
- 깔끔한 디자인과 다크모드, 폰 앱 설치(PWA)

1. 조우 시작 — W부터 묻히기
전투 시작하면 W(시너지) 먼저 깔고 시작. 그다음부터 2번 규칙대로.

----------

2. 평소 — 파란색 우선, 없으면 W
W부터 써서 시너지를 묻히고 →파란색(F·R·D·E) 쿨 도는 대로 계속 때리고, 쓸 파란색 없을 때만 W. 
T(초각스)는 쿨 뜨면 아무 때나 반사적으로.
파란색이 전부 쿨타임이라 스킬이 빌 때, A(보라색)나 Q(노란색)를 던져서 파란색 쿨 당겨오기.
S(보라색 피면기)는 웬만하면 사이클에 섞지 말고 무조건 피면용으로 아껴 쓰기

----------

3. 아덴 진입 전 — 시계바늘 6~9시에서 진입, F 털고 들어가기
시계바늘이 6~9시 사이일 때 진입이 베스트 (W가 두 번째로 돌아올 때쯤이면 대략 이 구간). 진입 직전 F 상태 보고:

  1) F 쓸 수 있으면 → 한 방 털고 Z
  2) F가 5초 이내로 남았으면 → Q/W/A/S 하나 껴서 쿨 당긴 뒤 F 털고 Z
  3) F가 한참 쿨이면 → 그냥 Z

시계바늘 12시 넘기면 손해 → F 털겠다고 짤스킬 욕심내다 넘길 것 같으면 억지로 안 털고 조금 앞당겨 진입. 6시 못 넘겼으면 E·R로 6시 넘기고 진입.

----------

4. 아덴 진입 후 — 파란색 우겨넣고 E·R 깔고 나오기
고배율 파란색(T·D·F) 먼저 털기 → 손 비면 노란색(Q·W)·보라색(A)으로만 잠깐 메꾸기(필수 아님) → 마지막에 E·R 깔고 Z 탈출.
S(피면기)는 패턴 피할 때 써야 함.

----------

⚠️ 5. 실수 대처 — 12시를 넘겨버렸다면?
이미 12시를 넘겼는데 파란색이 전부 쿨타임이라 쓸 게 없다면 미련 없이 그냥 Z 아덴 진입.
Z에서 탈출한 후 스킬이 꼬여 약간의 '현타'가 오지만 망한 것은 아님.
당황하지 말고 노란색·보라색 짤스킬을 물불 가리지 않고 묶어 던지며 쿨을 당겨오기. 
평소 사이클을 두 바퀴(W를 총 4번 쓸 때까지) 돌린다고 생각하시면 금방 원래 궤도로 복구됨.

----------

요약

1. 조우 땐 W부터 (시너지 묻히고 시작)
2. 평소엔 파란색(F·R·D·E) 쓰고, 없으면 W (빌 때 A·Q로 쿨 당기기 / S는 아껴서 피면용)
3. 시계바늘 6~9시에서 F 털고 Z 진입 (F가 5초 이내면 짤 껴서 쿨 당긴 뒤 털기 / 12시 넘기지 말 것)
4. 아덴에서 T·D·F 우겨넣고 → 마지막에 E·R 깔고 Z 탈출
5. 12시 넘겼으면 당황 말고 아덴 털고 → 노란색·보라색 묶어 던지며 두 바퀴(W를 총 4번)로 복구
